# Machine_Learning_and-Neural_Networks
The rapid growth of music streaming platforms has led to vast collections of songs, making it increasingly important to organise and analyse audio data effectively. Clustering techniques provide a useful way to group similar items, supporting applications such as recommendation systems, playlist generation, and user behaviour analysis. K-Means is one of the most widely used clustering algorithms due to its simplicity and efficiency. It partitions data into a predefined number of clusters based on similarity, typically measured using distance. However, its performance is highly dependent on key factors such as feature scaling and the selection of the number of clusters. This tutorial applies K-Means clustering to a Spotify tracks dataset, focusing on how feature scaling influences clustering performance and how the optimal number of clusters can be determined using the Elbow Method. In addition, Principal Component Analysis (PCA) is used to visualise the clustering results. The aim is to demonstrate how these techniques improve both the effectiveness and interpretability of clustering in real-world data.
